Banking Complaint & Fraud Management System (Backend)

A secure and scalable FinTech backend system built using the MERN stack to manage banking complaints and fraud reports with role-based access and document handling.

Key Features
JWT Authentication & Authorization
Role-Based Access Control (Admin / Staff / Customer)
Complaint Management
Register & track complaints
Status updates & routing
Fraud Reporting System
Document Uploads (Cloudinary)

Tech Stack
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Auth: JWT, bcrypt
File Upload: Cloudinary

Project Structure
models/
routes/
controllers/
middleware/
seeds/
server.js

⚙️ Setup & Run
git clone https://github.com/Aadil0513/Backend-bankComplaint.git
cd Backend-bankComplaint
npm install
npm run dev
